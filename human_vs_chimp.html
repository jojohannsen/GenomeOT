Genomes Over Time

Human vs. Chimp, one million base pairs per pixel

Input
  • FASTA files for each chromosome
  • Configuration file (see below)

Output
  • Two image files for each chromosome comparison, one at 100k bp per pixel, the other 1M bp per pixel
